## 🚀 Sobre o Projeto

O **GL Prime** é um site institucional desenvolvido para uma empresa do setor de **energia solar**, projetado para fortalecer a presença digital, apresentar projetos fotovoltaicos e automatizar o contato com clientes.

O projeto simula um ambiente real de desenvolvimento profissional, utilizando metodologias ágeis, arquitetura moderna em nuvem e divisão de papéis baseada em squads de tecnologia.
